Welcome to my Git cheatsheet

Einrichtung

git config --global user.name “[firstname lastname]”

Setze einen Namen für die Erwähnung in der Historie im Git.

git config --global user.email “[valid-email]”

Lege die E-Mail fest, welche in der Versionshistorie angezeigt wird.

Initialisierung des Git

git init

Eine bestehende Git-Repo initialisieren

git clone url

eine komplette Repo klonen

git status

modifzierte Dateien für den nächsten commit anzeigen

git add

Eine Datei zum commit hinzufügen. mit git add . kompletten Ordner & Dateien.

git commit -m "message"

Aktuelle Änderungen zum commit hinzufügen und mit einer Nachricht versehen

Branch & Merge

git branch

Listet alle branches auf. Die aktive branch ist mit * markiert

git branch branch-name

Eine neue branch mit "branch-name" erstellen

git checkout branch-name

In eine andere branch wechseln

git log

Zeige alle commits in der aktuellen branch Historie
